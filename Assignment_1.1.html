In final year my project was based on Iot that is Vehicle tracking n accident detection In which automatic alarm device for vehicle accidents has been implemented using AtMega328 microcontroller. This design is a system which can detect accidents in significantly less time and sends the basic information to first aid center within a few seconds.
The switch provides the driver a chance to cut off emergency help in case the system triggers a false alarm or if the accident is not very severe and immediate help is not required
